Meet Hedy!

The coding language made for kids

Multi-lingual Programming

Hedy supports over 47 different languages! Including Dutch, Spanish, Arabic, Turkish, Chinese and Hindi.

Built for the Classroom

Teachers can use our free built-in lesson plans, or create their own. Hedy is suitable for kids ages 10 and up.

Step by Step Learning

Hedy introduces concepts and syntax step by step. A scientifically proven way to learn!

Bridge the gap!

Hedy can be a bridge between programming tools like Scratch and textual programming like Python. It combines the friendliness of block coding with the power of real syntax to help students transition smoothly into textual programming.

Made for kids

Hedy is a textual programming language designed for kids in the classroom. For example, Hedy provides error messages that are easy for kids to read and understand. This helps students practice debugging on their own.

Hedy is free!

Hedy is “open source”, which means that everyone can help us make Hedy better. If you like Hedy and want to contribute, we accept (and are very grateful for) donations!

Hedy is an innovative programming language that helps learners transition from block-based coding (like Scratch) to text-based programming (like Python). It introduces syntax step-by-step, making learning easier and more fun.
